# markerai-design-skills

Personal Claude Code marketplace that bundles 7 design skills into one plugin (`markerai-design`):

- `ui-ux-pro-max` — UI/UX intelligence (50+ styles, 161 palettes, 57 font pairings, charts for 10 stacks)
- `design` — full design pipeline (logos, CIP, banners, icons, social photos, presentations)
- `design-system` — token architecture and component specifications
- `ui-styling` — shadcn/ui + Tailwind utility patterns
- `brand` — brand voice, tone, messaging frameworks
- `slides` — strategic HTML presentations (Chart.js, design tokens)
- `banner-design` — social/ads/web/print banners (22 styles)

## Install

```
/plugin marketplace add apexjutsu/markerai-design-skills
/plugin install markerai-design@markerai-design-skills
```
